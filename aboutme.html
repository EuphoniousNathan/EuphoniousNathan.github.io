What's HTML?

HTML known as "Hyper-Text Markup Language, HTML is a coding language that revolves around websites and graphics.
Every website is made from HTML and CSS (Cascading Style Sheets). HTML is used for the headers, paragraphs,
images, etc. CSS is used for touching up the webpage itself. For example, with CSS you have to figure out the
correct positioning for images, the correct font, the correct font size, etc. It can become a tedious and hectic
process at times, but in the end, things smoothen out when you get the hang of it.
